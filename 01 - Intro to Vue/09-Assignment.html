Assignment: Creating an Application with the CLI

Objectives:

  • Practice using the Vue CLI to create an application

To help solidify the concepts we've just discussed, we're going to convert our previous application into a Vue application using the CLI.

Note: Moving forward, always have your browser's console open, checking for any warnings or errors!

  1. Start by opening your terminal and navigating to where you are saving your Vue assignments.

  2. Then run the command, accepting the default option when prompted:

    vue create my-first-cli-app

  3. Once the application has been created, open the my-first-cli-app folder in your text editor.

  4. Replace the contents of the template, script, and style sections of the HelloWorld.vue file.

    Note: for this assignment, just the name is a variable

  5. Update the App.vue file as needed.
  6. Once you've got everything working as expected, change the name of the HelloWorld component to LearningList.
