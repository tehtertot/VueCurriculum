Nuxt Steps

Objectives:

  • Use SSR techniques in your very own app!

Use the star wars api, asyncData, and Nuxt to create a star wars character directory.

  • The directory displays at least 5 star wars characters in a list of your design.
  • The directory displays name, mass, hair color, eye color, and anything else you'd like to display.
  • BONUS: Display home planet name for each character.
