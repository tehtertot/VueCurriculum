Assignment: Login/Registration Part I

Objectives:

  • Create a new Vue application with Firebase
  • Set up a basic login/registration page to prepare for the next assignment

We aren't quite ready to make a completely functional login and registration page, but this assignment will set us up for integrating this functionality shortly. For this application, be sureto set up all the Firebase configurations as shown in the previous module.

Be sure to set up data variables for each of the input fields using v-model. While we don't have a backend ready to save usernames and passwords yet, we can still perform some front-end validations. With front-end validations, we can reduce unnecessary backend calls because we can check for basic requirements and notify the user if these are not meant. For example, if any required fields are missing, there is no reason to make a request to the backend.
