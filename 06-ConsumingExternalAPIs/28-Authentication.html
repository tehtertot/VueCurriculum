Authentication

Objectives:

  • Learn why authentication is important
  • Learn how to authenticate with Firebase

Up to this point, we haven't been concerned with who is interacting with our applications--anyone could edit, delete, or add new TV shows in our last application. While there may be times where this is desirable, more often we'll likely want to control what users are able to do and see, and so we need a way of knowing who a given user actually is.

As we develop web applications for users, authentication is a very important process to consider. Authentication is the process of verifying that someone is who they say they are. So far, none of our applications have utilized this process--anyone could walk up and play Ninja Gold or browse through the applications we've made; but suppose we did want it to matter. What if we wanted to keep track of Ninja Gold high scores or a history of game play per user? You have probably used dozens of sites that require you to authenticate by signing in with some combination of username and password or log in via an existing account like Google, Facebook, or GitHub.

While this is certainly something we could manage internally in our application (by saving usernames and encrypted passwords into a database and then checking that database each time a user logs in), we are going to utilize a third-party API, namely Firebase, to assist us with this process. We can trust a provider like Google to ensure our data is secure, and that way, we don't have to be experts in security; instead, we can focus on the heart of our application!

Firebase Setup

  1. Open your project in the Firebase console, if it isn't open already.
  2. On the left navigation bar, under "Develop," click "Authentication."
  3. In the main pane, select the "Sign-in method" tab.
  4. Under "Provider," click on Email/Password to edit it. Click the "Enable" toggle to turn this option on. (For now, leave the Email link (passwordless sign-in toggled off).

Vue App Setup

The Firebase Auth service has several useful methods that abstract away the complexities of login and registration. To follow along in the documentation, you can find it here.

  1. First, we'll need to create a registration form for users to enter an email and a password.

    <p>Email: <input type="text" v-model="email"></p>
<p>Password: <input type="password" v-model="password"></p>
<p><button v-on:click="registerUser">Register</button></p>

  2. Once a user has completed the form and clicks the "Register" button, we'll call the Firebase method to add this user:

    firebase.auth().createUserWithEmailAndPassword(this.email, this.password)

  3. If we go back to our Firebase console and look at the list of users, we should see the recently registered user!
