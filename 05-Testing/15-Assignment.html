Testing From Scratch

Objectives:

  • Practice writing unit tests from scratch

Write an app where a user plays rock, paper, scissors against a computer opponent. Your app should be well-tested.

  • Create a new Vue app using the CLI that adds Vuex, Vue Router, unit testing (using jest), and e2e testing (using cypress)
  • User can choose between rock, paper, or scissors.
  • Scissors beats paper, paper beats rock, rock beats scissors.
  • After each game, user can see their total wins and losses.
  • User has the option to start a new game.
  • Ninja Bonus: User can reset their score.

Hint: To create a random number within a range, you can use the following code: 

function createRandomInteger(min, max) {
  const range = max - min;
  return Math.round((Math.random() * range) + min)
}
