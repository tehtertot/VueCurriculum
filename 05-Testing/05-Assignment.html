Testing Fundamentals

Objectives:

  • Learn about when to use certain matchers
  • Write a simple test suite using just JS and Jest

Often times, when we're comparing objects in JavaScript like arrays and objects, we can't use the simple toBe matcher. We'll instead need to use toEqual. This is because objects and arrays are passed by reference, not by value. Therefore, comparing two arrays that seem identical, won't work because they are technically two separate arrays that contain the same values. To get around this, we can use the toEqual matcher for comparison.

Use the steps from the last few lessons along with the toEqual matcher to write a solution for the Array: Insert At challenge in the algorithm book and test it using a simple test suite.
